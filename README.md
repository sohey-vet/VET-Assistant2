# VET-Assistant2 - X投稿案自動生成ツール

獣医師向けX投稿案を自動生成する高精度デスクトップアプリケーションです。

## 🚀 特徴

- **高精度シーケンス制御**: 質問-回答ペア、ケース-解説ペアの確実な連続配置
- **重複回避システム**: 過去投稿との重複を完全回避
- **前後関係管理**: 症例に対する解説が必ず翌日に配置
- **指示書完全準拠**: 140文字以内、適切なフォーマット、専門性と親しみやすさの両立

## 📋 機能

- Xアーカイブのtweets.jsファイル自動解析
- 猫投稿（#猫のあれこれ）と犬投稿（#獣医が教える犬のはなし）の自動生成
- 3つの週テーマ：参加型、猫種特集、専門テーマ
- CSV形式での投稿案出力
- 重複チェック機能
- シンプルなGUIインターフェース

## 🛠️ インストール・セットアップ

### 初回セットアップ
1. `setup.bat` をダブルクリック（必要なライブラリを自動インストール）

### アプリケーション起動
- **通常版**: `run_complete.bat` をダブルクリック
- **高精度版**: `run_advanced.bat` をダブルクリック（推奨）

## 📖 使い方

1. アプリケーションを起動
2. tweets.jsファイルパスが自動設定済み
3. 生成日数を入力（デフォルト：7日/高精度版：14日）
4. 週テーマを選択
5. 「生成開始」ボタンをクリック
6. `output/` フォルダにCSVファイルが生成

## 📊 出力仕様

### ファイル形式
- **通常版**: `tweet_drafts_YYYY-MM-DD_[テーマ].csv`
- **高精度版**: `tweet_drafts_YYYY-MM-DD_[テーマ]_advanced.csv`

### 列構成
- `投稿日時`: YYYY-MM-DD HH:MM 形式
- `投稿内容`: 140文字以内の投稿テキスト
- `文字数`: 正確な文字数

### 投稿スケジュール
- **07:00**: 猫の投稿（#猫のあれこれ）
- **18:00**: 犬の投稿（#獣医が教える犬のはなし）

## 🎯 高精度版の特徴

### 前後関係管理
- 質問投稿 → 翌日に必ず回答投稿
- ケーススタディ → 翌日に必ず解説投稿
- シーケンス事前計画による確実な連続性

### 重複回避システム
- 過去投稿のMD5ハッシュ管理
- 数ヶ月分のコンテンツプール
- 完全な重複防止

### コンテンツプール
- 猫質問：30パターン + 対応回答
- 犬ケース：20パターン + 対応解説
- 健康投稿：豊富なバリエーション

## 📝 投稿例

### 猫投稿（参加型）
```
獣医師に教えて！愛猫のモーニングコールは？⏰

【質問】あなたの猫ちゃんは、どんな風に朝起こしてくれますか？
※コメントで番号を教えてね！

①顔の近くで鳴く
②顔や手を舐める・噛む
③体にのってくる
④その他（ぜひ具体的に！）

うちの猫様は、鼻を噛んできました😼
#猫のあれこれ
```

### 犬投稿（ケーススタディ）
```
【もしもの時… ケース⑥ 口のサイン】

症例：7歳のトイプードル。
最近、口臭が気になるし、硬いおやつを嫌がるように。
よだれも少し増えた気がする…🤤

このサインから、まず何を疑いますか？
(※あくまで架空の事例ですが、あるあるシチュエーションです)
#獣医が教える犬のはなし
```
